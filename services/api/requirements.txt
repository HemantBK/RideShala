# RideShala API — Core Dependencies
#
# This file contains ONLY what the API server needs to run.
# Heavy ML packages (torch, sentence-transformers) are NOT here.
# They go in pyproject.toml optional groups:
#
#   pip install -e ".[ml]"    → adds sentence-transformers (for local embedding/reranking)
#   pip install -e ".[eval]"  → adds ragas (for evaluation)
#   pip install -e ".[gpu]"   → adds vllm + sentence-transformers (for GPU serving)
#   pip install -e ".[dev]"   → adds pytest, ruff, mypy (for development)
#
# This separation ensures:
#   - Cloud deploys (Render/Railway) install in <30 seconds, not 10 minutes
#   - GPU machines install everything they need via optional groups
#   - CI installs only what's needed for testing
